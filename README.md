## 🚀 Running Locally

No build step required. You have two options:

### Option 1 — Open directly in browser

```bash
# Just open the file
open index.html
# or on Windows:
start index.html
```

### Option 2 — Use a local server (recommended, avoids CORS quirks)

**With VS Code (Live Server extension):**

1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
1. Right-click `index.html` → **Open with Live Server**

**With Python:**

```bash
# Python 3
python -m http.server 8080
# Then open: http://localhost:8080
```

**With Node.js (npx):**

```bash
npx serve .
```

## 🌐 Deploying

### Netlify (Recommended — free, zero config)

1. Push this folder to a GitHub repository
1. Go to [netlify.com](https://netlify.com) → **Add new site** → **Import from Git**
1. Select your repo — Netlify detects it as a static site automatically
1. Click **Deploy site**

### GitHub Pages

1. Push to a GitHub repo
1. Go to **Settings → Pages**
1. Set Source to `main` branch, folder `/` (root)
1. Your site will be live at `https://yourusername.github.io/repo-name/`

### Netlify Forms (contact form)

Add `data-netlify="true"` to the `<form>` tag in `index.html`:

```html
<form
  class="contact__form"
  id="contact-form"
  data-netlify="true"
  novalidate
></form>
```

Netlify handles submission automatically — no backend needed.

## ✏️ Replacing Placeholder Content

Search the codebase for `PLACEHOLDER` comments — every piece of temporary content is clearly marked.

### Common replacements:

| What to replace  | Where                                                    | What to change                  |
| ---------------- | -------------------------------------------------------- | ------------------------------- |
| Your name        | `index.html` → `<h1 class="hero__name">`                 | Replace `Kalophain Placeholder` |
| Your initials    | `index.html` → `hero__avatar-initials`                   | Replace `KP`                    |
| Your role/title  | `index.html` → `hero__role`                              | Replace the text                |
| Your bio         | `index.html` → `about__bio`                              | Replace the paragraph           |
| Profile photo    | `index.html` → `.hero__avatar`                           | Replace the div with `<img>`    |
| About photo      | `index.html` → `.about__image-placeholder`               | Replace div with `<img>`        |
| Project names    | `index.html` → `.project-card__title`                    | Replace each heading            |
| Project images   | `index.html` → `.project-card__image-placeholder`        | Replace div with `<img>`        |
| Project links    | `index.html` → `.project-card__overlay a`                | Set correct `href` values       |
| Skills           | `index.html` → `.skills__grid`                           | Edit/add `.skill-card` elements |
| Email address    | `index.html` → `contact__detail-value`, hero social link | Replace `placeholder@email.com` |
| Social URLs      | `index.html` → `.social-link` hrefs                      | GitHub, LinkedIn URLs           |
| CV / Resume      | `index.html` → `.hero__actions a[download]`              | Set `href` to your PDF path     |
| Footer year      | Auto-generated by `js/script.js`                         | No action needed                |
| Meta description | `index.html` → `<meta name="description">`               | Update with your actual content |
| Page title       | `index.html` → `<title>`                                 | Replace with your name          |
| Favicon          | `assets/images/favicon.ico`                              | Drop your favicon file here     |

## 🎨 Customising the Design

All design tokens are CSS custom properties in `:root` at the top of `css/style.css`:

```css
:root {
  --clr-accent: #f0a500; /* ← change the amber accent colour here */
  --clr-bg: #0d0d0f; /* ← main background */
  /* ... */
}
```

Change `--clr-accent` to instantly re-theme the entire site.
